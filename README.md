# Personal Portfolio Website

## Overview

This is a responsive personal portfolio website developed using HTML, CSS, JavaScript, and Bootstrap. The website showcases my profile, skills, projects, internship experience, certificates, and contact information in a professional and interactive format.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Professional portfolio layout
* Skills and experience section
* Project showcase section
* Internship certificates display
* Contact section
* Interactive user interface
* Modern and clean design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap

The objective of this project is to create a professional portfolio website that effectively presents personal skills, projects, achievements, and certifications while demonstrating frontend web development capabilities.
